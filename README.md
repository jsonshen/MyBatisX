# MyBatisX
A MyBatis generator extensions library

## Prerequisites

[Git][] and [JDK 8 or later][JDK8 build]

Be sure that your `JAVA_HOME` environment variable points to the `JDK1.8.0` folder
extracted from the JDK download.
